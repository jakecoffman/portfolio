gorunner

This is a project I am writing in Go. It's another continuous integration server like Jenkins. I was hitting a wall with Taskpy since Python doesn't do multi-threading well. Rather than picking up a Gevent or other coroutine library, I am rewriting in Go. Hey, it's an excuse to learn Go!

Job Management

You can create jobs, and bind tasks to them to be executed.

Write task scripts directly in the browser with syntax highlighting.
